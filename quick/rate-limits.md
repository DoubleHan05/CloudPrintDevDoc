# 限流与大小限制

::: warning 接入前请先评估
上线前务必确认业务场景不会触发限流，否则线上会出现 4015 / 6021 / 6027 等错误。
:::

## 接口级 QPS 限制

| 接口 | 单设备 QPS | 备注 |
|------|-----------|------|
| 所有打印接口（通用上限）| ≤ 6 次/秒 | 单台打印机 |
| `result`（结果查询）| ≤ 2 次/秒 | 同一 SN |
| `cloudServiceRecharge`（充值）| ≤ 10 次/秒 | 单 appId；同一 SN 60 分钟内只能充 1 次 |
| `batchStatus`（批量状态）| — | 单次 ≤ 100 台设备 |

## 单次请求大小限制

| 接口 / 参数 | 限制 | 说明 |
|-------------|------|------|
| `deviceWrite`（TSPL/ESC 指令）| ≤ 100 KB | 超出返回 6019 |
| `escWrite`（UP 系列 ESC）| ≤ 30 KB | 超出返回 6019 |
| `shareFilePrint`（共享云打印文件）| ≤ 150 KB | zlib 压缩后；超出返回 6029 |
| 图片 base64 | 建议 ≤ 50 KB | 过大影响打印速度 |
| `renderDataArray`（批量标签）| ≤ 50 张/次 | 超出返回 6022 |
| `tsplXmlWrite.jobs`（批量 XML）| ≤ 20 单；KMSX320 ≤ 5 单 | 超出返回 6024 |
| `result.jobIdStr`（结果查询）| ≤ 50 个 jobId/次 | 超出返回 6026 |
| `batchStatus.snsStr`（状态查询）| ≤ 100 台/次 | 超出返回 6025 |
| `broadcast.volumeContent`（语音）| ≤ 50 字符 | 超出返回 6036 |

## 共享云打印专属限制

| 限制 | 值 | 说明 |
|------|-----|------|
| 单设备每日打印量 | ≤ 1000 次 | 次日恢复 |
| 并发 | 同一设备同时只能有 1 个进行中任务 | 返回 6031 |

## 触发限流时的处理建议

1. **4015 签名错误** → 优先检查签名，不是限流
2. **6027 打印接口过频** → 本地排队，控制发送频率
3. **6021 result 查询过频** → 建议改用[异步回调](../http/task/callback)代替轮询
4. **6022 批量渲染过大** → 拆分为多次请求，每次 ≤ 50 张
