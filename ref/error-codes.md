# 错误码总表

::: info 说明
接口返回的 `code`（业务错误码）与 result / 回调里的 `code`（任务结果码）**不是同一套**，请勿混淆。
:::

## API 业务错误码

| code | 描述 | 处理方式 |
|------|------|----------|
| 2008 | 接口调用过于频繁 | 降低调用频次；充值接口请稍后重试 |
| 4000 | 序列号为空 | 检查 `sn` 是否传入 |
| 4001 | 序列号不存在或未激活 | 检查 SN 是否与机器一致；WiFi 机型确认蓝灯配网成功；4G 机型确认开机 |
| 4005 | 设备离线 | 检查网络状态 |
| 4009 | 设备密钥不正确 | 检查机器底部密钥与 `deviceKey` 一致 |
| 4012 | 无权限设备 | 确认设备已被该 appId 绑定 |
| 4015 | 签名错误 | 检查签名规则与 appSecret，参考[签名规则](/zh/quick/sign) |
| 4018 | 设备还未激活 | 确认设备已联网并完成激活后重试 |
| 4019 | appId 无可用充值券 | 联系平台开通或补充充值券 |
| 4020 | 剩余充值券不足 | 检查 `months` 是否超过可用月数 |
| 4021 | 卡号已销号，无法充值 | 更换设备或联系平台确认物联网卡状态 |
| 4022 | 充值失败 / 60 分钟内重复充值 | 60 分钟后再充；其他失败请联系平台 |
| 4023 | 充值结果未知 | 查询充值历史或联系平台确认，避免立即重复充值 |
| 4024 | 充值券配置异常 | 联系平台处理充值券配置 |
| 6006 | 必传参数为空 | 检查接口必传参数 |
| 6009 | 动态渲染数据结构错误 | 检查 renderData / renderDataArray 格式 |
| 6010 | 模版类型不匹配 | 标签模板用 tsplTemplatePrint；小票模板用 escTemplatePrint / tsplTemplateWrite |
| 6015 | 无权限访问模版 | 确认 templateId 属于该 appId |
| 6018 | 图片 Base64 数据错误 | 检查图片格式，建议 ≤ 50KB |
| 6019 | 打印内容过大 | deviceWrite ≤ 100KB；UP 系列 escWrite ≤ 30KB |
| 6021 | 该序列号查询过于频繁 | 同一 SN 每秒最多 2 次 |
| 6022 | 批量渲染数量过大 | renderDataArray 长度 ≤ 50 |
| 6024 | xml jobs 数量过大 | tsplXmlWrite.jobs ≤ 20；KMSX320 ≤ 5 |
| 6025 | 查询设备数超过限制 | batchStatus.snsStr ≤ 100 个 |
| 6026 | 打印结果查询超过限制 | result.jobIdStr ≤ 50 个 |
| 6027 | 打印接口调用过于频繁 | 单设备 QPS ≤ 6 |
| 6029 | 文件大小过大 | shareFilePrint 文件 ≤ 150KB（zlib 压缩后） |
| 6030 | 当前设备不是共享云打印机 | 使用打印管家生成的共享云打印序列号 |
| 6031 | 设备正在打印中 | 稍后再调用 |
| 6032 | 打印量超过当日上限 | shareFilePrint 单设备单日 ≤ 1000 次，次日恢复 |
| 6034 | 文件类型需为 zlib 压缩 | 共享云打印的 file 必须是 zlib 压缩 |
| 6035 | 该机型不支持语音播报 | 使用支持语音的机型（KM118MGL / KME31GP） |
| 6036 | 语音内容超过限制 | volumeContent ≤ 50 字符 |
| 6039 | 该机型不支持调节浓度 | 使用支持机型（KM118 / KME31 / KME41 系列） |

## 打印任务结果 code

下表 code 是 result 接口返回的 `data[].code`，与[异步回调推送](/zh/http/task/callback)里的 `code` 字段共用同一套。

| code | 描述 |
|------|------|
| 3000 | 下发成功（任务已到达打印机） |
| 2000 | 打印成功 |
| 2007 | 打印失败 |
| 2004 | 打印异常 |
| 3001 | 打印机模式错误导致失败 |
