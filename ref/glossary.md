# 术语汇总

## 核心凭证

| 术语 | 别称 | 说明 |
|---|---|---|
| appId | 应用 ID | 开放平台应用的唯一标识 |
| appSecret | 应用密钥 | 与 appId 配对，用于签名，严禁前端泄露 |
| sign | 签名值 | MD5 加密的签名字符串，32 位小写 |
| timestamp | 时间戳 | 请求时间，格式 `yyyy-MM-dd HH:mm:ss` |

## 设备标识

| 术语 | 别称 | 说明 |
|---|---|---|
| SN | 序列号 | 打印机唯一标识，贴于机器底部或自检打印页 |
| IMEI | 国际移动设备识别码 | KM360C 专用标识 |
| deviceKey | 设备密钥 | 部分设备机身贴有密钥，绑定 / 解绑时需要 |
| code | 激活码 | KM360C 绑定时由 getDeviceCode 获取，5 分钟有效 |
| share_ 前缀 SN | 共享云打印 SN | 番茄打印管家生成，用于 shareFilePrint 接口 |

## 打印任务

| 术语 | 别称 | 说明 |
|---|---|---|
| jobId | 任务 ID | 打印接口返回，用于查询结果或匹配回调 |
| templateId | 模板编号 | 模板编辑器中创建的模板 ID |
| renderData | 渲染数据 | 小票模板的动态数据（JSON 对象字符串） |
| renderDataArray | 批量渲染数据 | 标签模板的批量数据（JSON 数组字符串，每元素 = 一张标签） |
| bindTable / bindKey | 绑定表 / 字段 | 模板组件与数据 JSON 的对应关系（bindTable = 外层 key；bindKey = 数组对象内的字段） |

## 纸张与指令

| 术语 | 别称 | 说明 |
|---|---|---|
| 间隙纸 | 标签纸 | 每张之间有间隙，用 `tspl` 前缀接口 |
| 连续纸 | 小票纸 | 无缝连续卷纸，用 `esc` 前缀接口 |
| TSPL | 标签打印语言 | 间隙纸打印机的原生指令集 |
| ESC / ESC/POS | ESC 指令 | 连续纸打印机的原生指令集 |
| CPCL | 便携打印指令 | 蓝牙便携打印机的指令集 |
| DPI | 打印分辨率 | 每英寸打印点数，常见 203 / 300 |
| dot | 像素点 | 指令中的坐标单位，8 dot = 1 mm |

## 充值与配额

| 术语 | 别称 | 说明 |
|---|---|---|
| availableCount | 剩余月数 | appId 下剩余云服务充值券月数 |
| externalNo | 业务单号 | 对接方自定义充值单号，可重复 |
| localOrderNo | 本地单号 | 平台生成的充值单号，用于对账 |
| cloudServiceExpire | 云服务到期时间 | 4G 机型云服务过期时间 |
